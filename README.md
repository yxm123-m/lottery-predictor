# 🎱 彩票预测助手

基于统计学分析的彩票号码预测工具，支持 **体彩大乐透** 和 **福彩双色球**。

## 功能

- 🎯 多种预测算法（热门号码、冷门回补、加权随机、奇偶平衡）
- 📊 历史号码频率统计
- ⏳ 号码遗漏分析
- 🎨 精美号码球展示

## 在线使用

👉 [点击访问在线版本](https://your-app-url.streamlit.app)

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```
